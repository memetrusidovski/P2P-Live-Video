# 2. IP Subnet Penalties

A massive cluster of Sybil nodes run by a single attacker will often originate from the same cloud hosting provider or botnet block.

**IP Subnet Clustering Auditing:** 
The auditing node maps the physical IP addresses of the signers in the receipt bundle. If more than $20\%$ of the validating receipts originate from the same $/24$ IPv4 subnet or $/48$ IPv6 prefix, the entire receipt bundle's score is penalized by:
$$\text{PenaltyFactor} = 1 - \frac{\text{ClusteredIPs}}{\text{TotalIPs}}$$

This mathematically nullifies the value of thousands of fake PoU receipts generated by a single server farm.
